# React-Form
This is a simple React app that demonstrates how to use forms in React. The app allows users to enter their name, email address, and message and submit the form.


<h2>Technologies Used</h2>
  <ul>
    <li>React</li>
    <li>HTML/CSS</li>
    <li>JavaScript</li>
  </ul>
